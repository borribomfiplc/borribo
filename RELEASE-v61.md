# BORRIBO HRMS v61 — Mobile usability

## Mobile data views
- Added a complete mobile card view for Login Accounts on User Roles page.
- Added card views for branch attendance summaries, holidays, Fingerprint CSV preview, attendance reports, leave reports and monthly rankings.
- Every page that contains a desktop table now has a dedicated small-screen representation.

## Forms, dialogs and navigation
- Account audit/repair actions become full-width touch targets on small phones.
- Shared organization dialogs and operational dialogs behave as mobile bottom sheets with safe-area padding.
- Increased form control font size on phones to prevent iOS auto-zoom.
- Improved touch scrolling and overscroll behavior for wide content.
- Updated footer version to v61.

## Compact dashboard cards
- Reduced mobile stat-card padding and icon size.
- Hidden decorative sparkline charts on very narrow screens to prevent overflow.
